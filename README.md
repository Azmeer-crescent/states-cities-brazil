# Estados e Cidades do Brasil

Esta base foi criada com informações do [Instituto Brasileiro de Geografia e Estatística](http://www.cidades.ibge.gov.br/). Ao todo são **5570 cidades** nos **27 estados** brasileiros.

- CSV
- HTML
- JSON
- PHP
- SQL
